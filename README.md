# Blog

基于`.Net Core Web API`和`Vue`开发的博客系统，支持移动端和 PC 端显示（主打PC端，移动端显示效果不好）

预览网站，请访问 🌍[此网站](http://www.karldu.cn/)获取最新信息。

## Features

+ 博客
+ 想法
+ 读书笔记
+ 推荐信息
+ 微信平台
+ 机器人
+ 文件管理
