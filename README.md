# Blog

基于`.Net Web Api`和`Vue`开发的博客系统，主要功能有：博客、想法、推荐信息、友人帐、归档等功能。

支持移动端和 PC 端显示

查看网站，请访问 🌍[此网站](http://www.karldu.cn/)获取最新信息。

## 功能

+ 博客
+ 想法
+ 读书笔记
+ 推荐信息
+ 微信平台
+ 机器人
+ 文件管理
